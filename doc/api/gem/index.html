    A sortable value used to indicate relative positions of a set of associated elements.

    A sortable value used to indicate relative positions of a set of associated elements. Locations may be thought of as lists of arbitrary integers which are be compared element by element such that the first pair of values that differ determine the ordering of the Locations as a whole. If one Location is a proper prefex of another, then it sorts ahead of the other Location.
