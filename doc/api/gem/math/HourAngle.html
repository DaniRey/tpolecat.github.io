HourAngle 

Companion object HourAngle

final class HourAngle extends Angle

Exact hour angles represented as integral microseconds. These values form an Abelian group over addition, where the inverse is reflection around the 0-12h axis. This is a subgroup of the integral Angles where microarcseconds are evenly divisible by 15.
